Quick Reference Guide for Data Archivists (DRAFT)
 
================================

This Quick Reference Guide for Data Archivists provides data archivists with guidelines to document a micro-dataset in compliance with the Data Documentation Initiative (DDI) and the Dublin Core (DCMI) metadata standards, using the Microdata Management Toolkit provided by the International Household Survey Network (IHSN).
